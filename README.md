# Protein Structure Annotations

## About

## Set Up
1. 
```
pip3 install biopython xmlschema
```
2. Prepare a text file with PDB identifiers <pdbs_to_download.txt>

## Usage
1. 
```
python3 parse_sifts.py
```
2. 
```
python3 download_pdbs.py
```
3. 
```
python3 calculate_sasa.py
```
